# Stage 10E Route Trace, Boss Phases And UI Fixes

# Chrono Echo: Fractured Realms

## 1. Purpose

针对试玩反馈：

1. 时间树看不出已走过的路线。
2. 精英与 Boss 攻击手段单一。
3. 商店卡片说明文字超出卡框。
4. 通关结算页文字重叠（结算理由与统计首行叠加）。

## 2. Scope

本阶段实现：

- 路线连线：时间树按 `selectedNodeIds` 顺序把已经过的节点用青色折线连接，绘制在节点圆之下，一眼可见本局路线。
- Boss 阶段与攻击扩展：
  - Fracture Warden：生命低于 50% 进入激怒阶段——扇形弹从 3 连发变 5 连发，且每第三轮攻击改为蓄力冲锋（450ms 前摇脉冲提示后高速冲向玩家）。
  - Glitch Warden：激怒阶段环形弹从 8 发增至 12 发，且每次环形弹幕前瞬移到玩家附近（带落点脉冲提示）。
  - 精英房强化：精英房内 Shooter 改为 3 连扇形弹，Anomaly 落点连射 2 发。
- 商店/奖励卡片扩容：卡片从 260x240 扩到 272x292，标题、描述、按钮行重新布局，中西文长描述不再溢出。
- 结算页重排：统计块与残响详情改为顶部锚定并动态排布，理由行与统计块不再重叠。

## 3. Design Rules

- Boss 新招式都有前摇提示（脉冲），保持可读可躲。
- 冲锋与瞬移可被 Time Freeze 打断（冻结时计时暂停）。
- 路线连线使用低饱和青色，不与可选节点的呼吸脉冲争夺注意力。

## 6. Hotfix: CJK Word Wrap

首轮修复后玩家反馈商店卡片描述仍横向溢出。根因：Phaser 默认换行只按空格分词，中文整句没有空格会被当作单个超长词而无法折行，卡片扩容无法解决横向溢出。上一轮验证只检查了垂直边界，遗漏了横向。

修复：`main.ts` 的全局文本工厂补丁对所有带 `wordWrap.width` 的文本默认启用 `useAdvancedWrap`（按字符折行），一次性覆盖全部场景与未来新增文本。

复验：中文商店三张卡片全部文本横纵边界均在卡框内（零违规），事件页零横向溢出，冒烟测试全部通过。
